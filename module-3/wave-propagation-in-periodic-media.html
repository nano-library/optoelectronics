Normal Modes of Propagation

✦
\(\vec{E}\) = \(\vec{E}_k \; e^{{\color{#F08FB0}{ - }} i \, \vec{k} \, \vec{x}}\)
where
\(k\) - block wave vector
\(\vec{E}_k\) - periodic amplitude
✦ Equation (13.6)
\(\vec{H}\) = \(\vec{H}_k\) \(e^{-i \; \vec{k} \; \vec{x}}\)
where
\(k\) - block wave vector
\(\vec{H}_k\) - periodic amplitude

Periodic Structure

✦
\(\varepsilon {\color{#F08FB0}{ ( }} z {\color{#F08FB0}{ ) }} \) = \(\varepsilon \: {\color{#F08FB0}{ ( }} z \; {\color{#F08FB0}{ + }} \; \Lambda {\color{#F08FB0}{ ) }}\)
✦
\(n {\color{#F08FB0}{ ( }} z {\color{#F08FB0}{ ) }} \) = \(n \: {\color{#F08FB0}{ ( }} z \; {\color{#F08FB0}{ + }} \; \Lambda {\color{#F08FB0}{ ) }}\)
where
\(\Lambda\) - permittivity

Dispersion Relation

✦
\(\omega\) = \(\omega \: {\color{#F08FB0}{ ( }} \vec{k} {\color{#F08FB0}{ ) }}\)
where
\(k\) - complex parameter

The wave vector \(\vec{k}\) determines the wave's behavior in the medium:

  • If \(\vec{k}\) is real, the wave propagates with only a phase change.
  • If \(\vec{k}\) is imaginary, the wave amplitude decays exponentially.
  • The range of frequencies \(\omega\) for which \(\vec{k}\) is imaginary is called the Photonic Band Gap.

Diffraction Condition in Periodic Media

Suppose \(\Lambda\) is spacing between planes. Path difference of rays reflected from two planes is \(2 \lambda \sin \theta\).
✦
\(2 \, \Lambda \, {\color{#F08FB0}{sin({\color{#6AAEE0}{\theta}})}}\) = \(m \, \lambda\)
where
\(2 \, \Lambda \, \sin(\theta)\) - physical path
✦
\(n \,2 \, \Lambda \, {\color{#F08FB0}{sin({\color{#6AAEE0}{\theta}})}}\) = \(m \, \lambda\)
where
\(n\) - average refractive index
\(n \, 2 \, \Lambda \, \sin(\theta)\) - optical path, that is in order of wavelength
✦ Diffraction Condition
✦
\(2 \, \Lambda \, {\color{#F08FB0}{sin({\color{#6AAEE0}{\theta}})}}\) = \(m \, \lambda\) \(n\)
